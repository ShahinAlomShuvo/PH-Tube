Discuss the scope of var, let, and const ?

Ans: The var Keywords are used to declare function scope and global scope,var keyword can not declare block scoped.
Variable which are declared by var keywords that are hoisted at the top.
Variables declared with var can be reassigned and redeclare within the same scope.

The let Keywords are used to declare block scope.
let can be reassigned within the same scope, but let can not be redeclare in the same scope.

The const Keywords are used to declare block scope,just like let.
const must be assigned a value when it's declared, and once assigned, it's value cannot be changed.

Tell us the use cases of null and undefined ?

Ans: The null and undefined are special values in JavaScript null and undefined represent the absence of a value.

The undefined is a primitive value that represents the absence of a value.
Variables that are declared but not assigned a value automatically have the value undefined.
Functions that do not return a value thats function are returns undefined.

The null is also a primitive value that represents the intentional absence of any value or an empty value.
It can be used to reset or clear the value of a variable or object property.

What do you mean by REST API?

Ans: A REST API is an interface that lets applications communicate over the internet using standard HTTP methods, such as GET, POST, PUT, and DELETE. It allows clients to access and manipulate resources on a server using URLs and is widely used for building web services.
